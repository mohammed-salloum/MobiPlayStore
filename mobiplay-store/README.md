# MobiPlayStore

متجر إلكتروني لشراء بطاقات شحن ألعاب الموبايل الشهيرة.

## التقنيات المستخدمة

- **React:** لبناء واجهة المستخدم بطريقة مكونية (Component-Based) وتحديث الواجهة بشكل ديناميكي وسلس.
- **Vite:** أداة تطوير سريعة وحديثة لتجميع المشروع وتشغيله مع دعم Hot Module Replacement (HMR) لتحسين تجربة التطوير.
- **JavaScript (ES6+):** لغة البرمجة الأساسية لإنشاء وظائف الموقع.
- **React Router:** لإدارة التنقل بين صفحات المتجر بشكل سلس وبدون إعادة تحميل الصفحة.
- **Redux (أو Context API):** لإدارة حالة التطبيق بطريقة منظمة، خاصة عند التعامل مع بيانات المستخدم أو عربة الشراء.
- **Bootstrap:** إطار عمل CSS لتصميم واجهة متجاوبة وجميلة بسرعة وسهولة.
- **ESLint:** أداة لفحص جودة الكود والتأكد من الالتزام بمعايير البرمجة.
- **Axios أو Fetch API:** للتواصل مع الخوادم والحصول على بيانات المنتجات أو إرسال الطلبات.
  
## طريقة التشغيل

1. تثبيت الحزم: `npm install`  
2. تشغيل السيرفر التطويري: `npm run dev`
